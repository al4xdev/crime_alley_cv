# Senior GenAI Platform Engineer — TechNexus Solutions

## Position Summary

TechNexus Solutions is seeking a Senior GenAI Platform Engineer to design, scale, and optimize our next-generation Generative AI orchestration platforms. In this role, you will build the core runtime systems, secure execution sandboxes, and agent communication loops that power our automated software-development agents.

---

## Key Responsibilities

1.  **Orchestration & Infrastructure**: Design, build, and optimize multi-agent feedback loops and prompt-runbook compilers without relying on heavy external frameworks.
2.  **Security & Sandboxing**: Implement strict, multi-layer physical sandboxing boundaries (utilizing Podman/Docker, nested namespaces, and volume controls) to safely execute untrusted candidate/agent code.
3.  **Cost & Token Optimization**: Implement Gemini Context Caching, global file filtering, and token tracking mechanisms to ensure cost-efficient LLM operation at scale.
4.  **Deterministic Controls**: Establish deterministic state tracking, exit condition checking, and log carry-forward pipelines to guarantee system reproducibility and prevent silent execution drifts.

---

## Required Technical Qualifications

*   **Experience**: 5+ years of software engineering experience focusing on backend systems, platform engineering, or developer tooling.
*   **Containers & Security**: Deep knowledge of container runtimes (Docker/Podman), Linux cgroups/namespaces, and rootless execution models.
*   **Programming**: Advanced proficiency in Python and Shell scripting (Bash/Fish). Experience with Go is a plus.
*   **LLM Integration**: Practical experience working with LLM APIs (e.g., Gemini API), context caching strategies, and telemetry monitoring.
*   **Testing**: Commitment to test-driven development (TDD) and writing reliable unit tests.
